# verify-number
WebComponent that provides opportunity to verify number.
## Usage
### Installation
```sh
npm i verify-number
```
### In  HTML file
```html
<html>
  <head>
  </head>
  <body>
    <verify-number mask="+7(985)0II-**-**" errortext="Wrong number"></verify-number>
    <script type="text/javascript" src="index.js"></script>
  </body>
</html>
```
### In  your index.js
```typescript
import 'verify-number';
```
